*[Установка TS и запуск файлов (6)]*

? 6.0 В этом уроке мы установим TypeScript, а также NPM-библиотеку "TS Node" для компиляции TS-файлов вручную и посмотреть, что из них получается в итоге.

Перейдём сразу на файл с примером: [06-test.ts]

// ? 6.1.1 При установке TypeScript на компьютер вместе с ним устанавливается и компилятор. Для его вызова на конкретном файле мы можем в консоли набрать команду `tsc path-to-file/file.ts` (перечень доступных команд вызывается командой `tsc --help`).
// ? 6.1.2 команда `tsc --init`, которая создаёт базовый шаблон настроек в проекте в файле "tsconfig.json".

// ? 6.2.0 Для более удобной компиляции файлов TS (одной командой, а не сначала `tsc file-name.ts`, затем `node file-name.js`) мы используем библиотеку «TS Node», запускаемую командой `ts-node file-name.ts`.
// ? 6.2.1 Кстати, если запустить команду `ts-node` без указания файла, то мы попадаем в режим "REPL" (Read-event-print loop), это такой режим Node, где можно вводить различные JS/TS команды и терминал будет сразу же выдавать результат. Выйти из него можно всё тем же "Ctrl" + "C", нажав их дважды. (Больше информации по использованию "TS Node" в режиме "REPL" в документации к нему по ссылке внизу.)
